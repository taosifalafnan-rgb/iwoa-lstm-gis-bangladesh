## What This Project Does

1. Downloads and preprocesses Landsat satellite imagery (2000вЂ“2024) for Gazipur and Narayanganj
2. Computes spectral indices: NDVI, NDBI, MNDWI, LST, UI
3. Runs LULC classification (Random Forest + SVM ensemble)
4. Performs spatial hotspot analysis (Getis-Ord Gi*)
5. Runs IWOA (Improved Whale Optimization Algorithm) to select optimal features
   and tune LSTM hyperparameters
6. Trains a multi-output LSTM to predict AQI, WQI, LST, NDVI monthly to 2040
7. Computes Human Habitability Index (HHI) using AHP weights
8. Produces publication-grade output maps and figures

## Pipeline Execution Order

Run these in sequence. Each step saves its outputs to `data/processed/`
so subsequent steps can load them without re-running earlier steps.

```
Step 1:  python src/data/loader.py          в†’ validates raw data, reports missing
Step 2:  python src/data/preprocessor.py   в†’ cleans, imputes, normalizes
Step 3:  python src/data/indices.py         в†’ computes spectral indices from rasters
Step 4:  python src/data/sequencer.py       в†’ builds LSTM sequence tensors
Step 5:  python src/optimization/iwoa.py    в†’ feature selection + hyperparameter tuning
Step 6:  python src/models/trainer.py       в†’ trains final LSTM
Step 7:  python src/models/evaluator.py     в†’ test set metrics + baseline comparison
Step 8:  python src/models/forecaster.py    в†’ 2025вЂ“2040 scenario projections
Step 9:  python src/analysis/ahp.py         в†’ AHP weights + CR validation
Step 10: python src/analysis/hhi.py         в†’ HHI computation per ward per year
Step 11: python src/visualization/plots.py  в†’ all figures
Step 12: python src/visualization/maps.py   в†’ all choropleth maps
```

Or run the full pipeline at once:
```bash
python run_pipeline.py --config configs/config.yaml
```

## Data Drop Instructions

When you receive your datasets, drop them as follows:

### DoE AQI Data в†’ `data/doe/`
Expected filename format: `doe_aqi_YYYY.csv` or `doe_aqi_gazipur_YYYY.csv`
Required columns: `date, station, pm25, pm10, no2, so2, co, aqi`
Date format: `YYYY-MM-DD`

### BWDB Water Quality в†’ `data/bwdb/`
Expected filename format: `bwdb_waterquality_YYYY.csv`
Required columns: `date, station, river, bod, do, ph, turbidity`
Date format: `YYYY-MM-DD`

### Satellite Rasters в†’ `data/satellite/`
Expected filename format: `L5_YYYYMM_B{band}.tif` for Landsat 5
Coordinate system: WGS84 (EPSG:4326) or Bangladesh Transverse Mercator (BTM)
Bands: B3 (Red), B4 (NIR), B5 (SWIR1), B6 (Thermal), B7 (SWIR2), QA_PIXEL

### Census Data в†’ `data/census/`
Expected filename: `bbs_population_ward.csv`
Required columns: `ward_id, district, upazila, year, population, area_km2`

### CHIRPS Precipitation в†’ `data/satellite/chirps/`
Expected: Monthly NetCDF or GeoTIFF files named `chirps_YYYYMM.tif`

### ERA5 Climate в†’ `data/satellite/era5/`
Expected: Monthly CSV with columns `date, temp_mean, humidity_mean, wind_speed`

## Configuration вЂ” configs/config.yaml

All parameters are controlled here. Never change parameters inside source files.
Key sections:
- `study_area`: bounding box, district names, CRS
- `data`: file paths, date ranges, expected columns
- `preprocessing`: imputation method, normalization, lag windows
- `iwoa`: population size, max iterations, Levy lambda, OBL flag
- `lstm`: architecture defaults (overridden by IWOA output)
- `training`: epochs, patience, batch size, loss function
- `hhi`: AHP matrix, sub-index weights, vulnerability thresholds
- `scenarios`: BAU/S1/S2 growth rate assumptions
- `wandb`: project name, entity, enabled flag
- `seed`: global random seed

## Key Technical Decisions

| Decision | Choice | Reason |
|---|---|---|
| Framework | PyTorch | Flexibility for custom IWOA-LSTM coupling |
| Loss function | Huber | Robust to pollution spike outliers |
| Optimizer | AdamW | Weight decay prevents overfitting on small dataset |
| LR schedule | Cosine annealing | Smooth convergence for 200-288 row dataset |
| CV strategy | Temporal block K-fold | Prevents data leakage across time |
| Uncertainty | MC Dropout (T=50) | Lightweight inference-time uncertainty |
| Spatial weights | KNN (k=8) | Handles uneven DoE station spacing |
| AHP scale | Saaty 1-9 | Standard, reviewer-accepted |
| Normalization | Min-Max to 0-100 | Required for HHI weighted sum |
